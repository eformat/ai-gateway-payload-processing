# AI Gateway Payload Processing

This repository contains BBR (Best of Breed Routing) plugins for the [llm-d](https://github.com/llm-d) AI Gateway inference scheduler.

BBR plugins enable custom request routing logic for LLM inference workloads, allowing fine-grained control over how requests are distributed across model-serving backends.

## Related Projects

- [llm-d-inference-scheduler](https://github.com/llm-d/llm-d-inference-scheduler) — The inference scheduler (EPP) that hosts BBR plugins
- [semantic-router](https://github.com/opendatahub-io/semantic-router) — vLLM Semantic Router (vSR) for semantic-aware routing
